круд:
1. сохранить новую книгу +++
2. получить книгу по id +++
3. редактировать книгу +++
4. удалить книгу +++
5. показать количество книг в базе +++
6. показать список всех книг в базе +++

1. главная страница:
    1. сверху отображаем количество книг в базе (5)
    2. ниже кнопка добавить книгу (1)
    3. ниже отображается список книг (6), у каждой книги есть кнопка редактировать (2 + 3) и удалить (4)

2. страница добавления:
    1. поля для ввода данных (пока что тогда все данные пусть записываются каждый раз, т.е. без всяких выпадающих
       списков авторов и жанров, потому что тогда надо еще делать круд на авторов и жанры)
    2. кнопка создать

3. страница редактирования:
    1. поля с заполненными данными книги, id заблокирован
    2. кнопка сохранить
